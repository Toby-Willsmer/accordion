Accordion for desktop and mobile

An accordion for general full width use & one that only appears on designated mobile breakpoint size.
Just resize your browser to see it in action.

You can use one of the three js files included, to either:

  1. Accordion all sizes accordion_1.js
  2. Accordion on mobile only accordion_2.js
  3. Both together accordion_3.js

For the purpose of this demo this page loads 'accordion_3.js'.
